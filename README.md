# Manroop Singh Portfolio

A modern personal portfolio website built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Customize

- Update portfolio content, links, skills, experience, projects, education, and SEO in `data/portfolio.json`.
- Replace `public/images/developer-workstation.png` if you want a different hero visual.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
